1. Introduction

About BiDaR

BiDaR stands for Big Data Retrieval. It is a smart micro-service-based web system that is able to retrieve information from DBpedia in multiple formats. BiDaR offers four main functionalities, search for a topic, visualization of demographics statistics per countries, a recommendation system for receipts base on user ingredients and statistics about all games played around the world. The main gamesoal is to create complex SPARQL queries for different areas of interest.

2. Using the website

Login and register page

To use BiDaR you need to login with email and password if an account already exists, if not than you need to create one on register page. The register page can be accessed by clicking register button.

For registration there are a few detalis that need to be filled like email address, password, name, gender, birthday and state. Remember that email is unique in our system, so you can't add an email already used and all fields need to be compeleted.

Profile page

All the details filled by user can be seen in profile page, where also can be updated by changing the values from every text box and sent them via update button. Also, in this page an user can add a topic of interest from the dropdown list with the add button and can delete a topic by pressing the label with the name of interest.

Search page

In this page you can type a topic into search bar and vizualize the rdf graph resulted. Also the rdf result can pe downloaded in any format by the download button.

Recommender page

An user need to choose from the ingredient list one or more ingredients to receive a recommendation. This can be visualized like a table if user clicked on Get Recommendation button or like an image in Word Cloud format when the button "Word Cloud" is pressed.

Demographics page

Demographics page offers six funtionalities, for all of them you need to choose first countries that are you interested in from first dropdown list. In second dropdown are two services "Population, life expectancy, HDI" and "Fertility, democracy, life expectancy", based on what user choose from this list the third dropdwon will update. If the first service is choosed then in third list will appear these three information: "Population throughout years" which shows a stack plot, "Life expectancy throughout years" which returns a lineplot and "HDI" which shows a pieplot. If the second service is choosed then, from the third dropdown list, the user will have to pick between maximum, minimum and average options. Also under this list will appear three text boxes where the user should introduce a value to compare each parameter.

Games page

As user you can see in this page three statistics: the distribution of games per countries, the distribution of games per continents, and comparation of games between two countries. Also, exists two lists one with counties and one with continents that allow user to exclude the selected ones from statistics.
